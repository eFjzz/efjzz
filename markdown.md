# ***Разметка Markdown***

## Что такое Markdown:

**Markdown** — это лёгкий язык разметки, который используется для форматирования текста.  
+ Он позволяет писать структурированный текст (заголовки, списки, ссылки, код и т.д.) в простом текстовом формате.

> Используется в README файлах (GitHub) ; документации ; блогах ; заметках ; системах управления контентом ;
---
## Заголовки:
Количество `#` определяет уровень заголовка:

# Заголовок 1
## Заголовок 2
### Заголовок 3
#### Заголовок 4
##### Заголовок 5
###### Заголовок 6
---
## Выделение текста:

**Жирный текст**

> ** текст **

*Курсив*

> * текст *

***Жирный курсив***

> *** текст ***

~~Зачёркнутый текст~~

> ~~ текст ~~
---
## Списки:
+ Маркированный список

-(+) Первый пункт

-(+) Второй пункт
  
  -(+) Подпункт

(12345) Нумерованный список

1. Первый
2. Второй
3. Третий
---
## Ссылки:
+ Встроенная ссылка в текст:
Example

> { текст ссылки ] ( example.com )

+ Встроенная ссылка в изображения:
Png

> !{ Описание изображения ]( image.png )
---
## Код:

Используй команду `ls -la`

> Используй команду 'ls -la `
