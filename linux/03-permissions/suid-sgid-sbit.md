# Linux Permissions Cheat Sheet
---
## 1. Права доступа к каталогам

| Символ | Значение | Каталог |
|--------|----------|---------|
| r      | read     | просмотр содержимого (`ls`) |
| w      | write    | создание/удаление файлов |
| x      | execute  | вход в каталог (`cd`) |

**Пример:**
`chmod 755 mydir`

`ls -ld mydir`

`drwxr-xr-x 2 user group 4096 (time) mydir`

---

## 2. Special Permissions:
| Permission  | Символ | Число | Что делает |
|------------|--------|-------|------------|
| SUID       | s      | 4     | запуск файла с правами владельца |
| SGID       | s      | 2     | запуск с правами группы / наследование группы для новых файлов |
| Sticky Bit | t      | 1     | удалять файлы может только владелец |

> Установка:

# SUID:
`chmod u+s file` or `chmod 4755 file`

# SGID:
`chmod g+s dir/file` or `chmod 2755 dir`

# Sticky Bit:
`chmod +t dir` or `chmod 1777 dir`

> SUID: только для бинарных файлов, которые требуют повышенных прав

> SGID: удобно для совместных каталогов

> Sticky Bit: используйте для публичных папок, чтобы никто не удалял чужие файлы.

---

## Примеры вывода `ls -l`

| Пример       | Описание |
|--------------|----------|
| `-rwsr-xr-x` | SUID установлен на файл |
| `drwxr-sr-x` | SGID установлен на каталог |
| `drwxrwxrwt` | Sticky Bit установлен на каталог (`/tmp`) |

## Добавить/убрать права
| Команда            | Права |
|--------------------|---------|
| `chmod u+rwx file` | # права владельца |
| `chmod g+rw dir`   | # права группы |
| `chmod o+r file`   | # права других |
