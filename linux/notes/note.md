## Обозначения 
+ имя файла - отдельная конструкция, которая нужна для более понятного обращения к пользователю.
+ директория - тоже является файлом, который содержит внутри себя другие файлы.
+ типы файлов - 
  
  > это файлы данных
  
  > это файлы являющиеся каталогами
 
  > специальные файлы устройств и т.п.

# Просто знать, какие типы документов существуют
> текстовые ( txt, docx )
> изображения ( jpg, png, svg )
> видео ( mp4, avi )
> аудио ( mp3, wav )
> исполняемые ( exe )
> архивы ( zip,rar )
