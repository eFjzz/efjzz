# Notes:
---
## Структура директорий `Linux` — это иерархия, где всё начинается с корня ( / ). 
+ Основные папки отвечают за разделение исполняемых файлов (/bin), настроек (/etc), данных пользователей (/home), устройств (/dev) и виртуальной информации о процессах (/sys, /proc). 
> Это обеспечивает безопасность и порядок в системе. 
---
### Linux Filesystem Structure

| Директория | Назначение |
|------------|------------|
| /home      | файлы пользователей |
| /etc       | конфигурационные файлы |
| /var       | логи и изменяемые данные |
| /usr       | программы и утилиты |
| /bin       | базовые команды |

---
### Основные папки ( с объяснением ):
+ `/usr` (User System Resources): Пользовательские программы, библиотеки и документация (большая часть ПО).
+ `/home` (Home): Личные каталоги пользователей (документы, загрузки, рабочие столы).
+ `/bin` (Binaries): Основные исполняемые файлы (команды: ls, cd, cat), нужные для работы системы даже в режиме восстановления.
+ `/etc` (Et cetera): Конфигурационные файлы системы и установленных программ (настройки сети, пользователей, служб).
+ `/dev` (Devices): Файлы устройств, представляющие компоненты компьютера (жесткие диски: /dev/sda, терминалы: /dev/tty).
+ `/sys` (System): Виртуальная файловая система, предоставляющая информацию о ядре, драйверах и аппаратном обеспечении.
+ `/snap` : Каталог для хранения приложений, установленных через систему пакетов Snap (популярно в Ubuntu).
+ `/cdrom` : Точка монтирования для CD-ROM/DVD дисков (сейчас чаще используется /media или /mnt).
+ `/root` : Домашняя директория суперпользователя (root), отдельная от остальных пользователей.
+ `/var` (Variable): Изменяемые данные: логи (/var/log), кэши, базы данных.
+ `/lib` (Libraries): Системные библиотеки, необходимые для работы программ из /bin и /sbin. 
---
### Файлы:

+ `имя файла`: отдельная конструкция, которая нужна для более понятного обращения к пользователю.

+ `директория`: объект файловой системы, служащий для организации файлов и других вложенных директорий.

+ `типы файлов`: 

> это файлы данных
  
> это файлы являющиеся каталогами

> специальные файлы устройств и т.п.
