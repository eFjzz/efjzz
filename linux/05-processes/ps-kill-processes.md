# basic.processes

---

## Базовая работа с процессами включает:
+ просмотр процессов (ps, top)
+ поиск процессов (pgrep, grep)
+ завершение процессов (kill, killall)
+ управление фоновыми задачами (jobs, fg, bg)

> где

| Команда |Назначение|
|---------|----------|
| `ps`    | - список процессов; |
| `top`   | -	мониторинг процессов; |
| `kill`  | - завершить процесс; |
|`killall`| -	завершает все указанные процессы по имени программы; |
| `pgrep` | -	найти PID процесса; |
| `jobs`  | - управление фоновыми задачами; |

---

# Процесс 

> это запущенная программа в системе Linux.

### Каждый процесс имеет:

**PID** — уникальный идентификатор процесса
**PPID** — идентификатор родительского процесса
**USER** — пользователь, который запустил процесс
**CPU / MEM** — использование ресурсов

---

## Просмотр процессов:

+ `ps` — список процессов
> показывает процессы, запущенные в системе.

### Полезная комбинация:
`ps aux`

|Ключ|Значение|
|----|--------|
|`-a`| процессы всех пользователей |
|`-u`| подробная информация | 
|`-x`| процессы без терминала | 

> если ввести все значения, покажет процессы исполняемые из под root'a.

---

## Мониторинг процессов: 
`top`
> показывает процессы в реальном времени.

### (top) Отображает:

+ загрузку CPU
+ использование RAM
+ активные процессы

### Горячие клавиши:

|Клавиша|Действие|
|-------|--------|
|  `q`  | выйти |
|  `k`	| завершить процесс |
|  `M`	| сортировать по памяти |
|  `P`  | сортировать по CPU | 

> действия совершаются в самом окне процессов.

---

## Завершение процессов:
`kill PID`
> завершает процесс по указанному индетификатору процесса.

+ Пример:
`kill 2541`

+ Принудительное завершение:

`kill -9 PID`

## Завершение назначается по нескольким ключам:
+ -1 --> перезапуск процесса
+ -9 --> принудительное завершение (SIGKILL)
+ -15	--> корректное завершение (SIGTERM)
+ -u --> завершить процессы конкретного пользователя
+ -v --> показать какие процессы были завершены

---

### Управление заданиями:

Запуск программы в фоне:
`command &`

+ Пример:
`firefox &`

+ Показывает фоновые задачи:
`jobs`

+ Вернуть процесс в передний план:
`fg %1`

+ Продолжить выполнение в фоне:
`bg %1`
