# Vim:

## Что такое Vim:

**Vim (Vi IMproved)**
> это мощный консольный текстовый редактор, являющийся улучшенной версией классического редактора `vi`. 

> Он используется разработчиками, системными администраторами и продвинутыми пользователями для быстрого и эффективного редактирования текста.

---

## Основные особенности Vim:
+ Работа в терминале
+ Высокая скорость редактирования
+ Управление с клавиатуры без мыши
+ Расширяемость через плагины
+ Поддержка множества языков программирования
+ Кроссплатформенность (Linux, macOS, Windows)

## Режимы работы Vim
+ Vim работает в нескольких режимах:

---

### Normal Mode (Обычный режим):
> Режим по умолчанию. Используется для навигации и выполнения команд.

- `h` `j` `k` `l` — перемещение курсора
- `dd` — удалить строку
- `yy` — копировать строку
- `p` — вставить
- `u` — отменить действие

---

### Insert Mode (Режим вставки):
> Используется для ввода текста.
+ Переход:
- `i` — вставка перед курсором
- `a` — вставка после курсора
- `o` — новая строка ниже

Выход:
- `Esc`

---

### Visual Mode (Визуальный режим):
> Используется для выделения текста.

- `v` — символьное выделение
- `V` — выделение строк
- `Ctrl + v` — блочное выделение

---

### Command Mode (Командный режим):
> Для выполнения команд сохранения, выхода и настроек.

Примеры:
- `:w` — сохранить
- `:q` — выйти
- `:wq` — сохранить и выйти
- `:q!` — выйти без сохранения
- `:set number` — включить нумерацию строк

---

## Полезные команды

| Команда | Описание |
|----------|----------|
| `gg` | Перейти в начало файла |
| `G` | Перейти в конец файла |
| `/text` | Поиск текста |
| `:%s/old/new/g` | Замена по всему файлу |
| `ciw` | Заменить слово |
| `dw` | Удалить слово |
