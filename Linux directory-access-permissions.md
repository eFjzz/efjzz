# Избирательные права доступа к каталогам (Linux)

---

+ В Linux доступ к каталогам регулируется тремя типами прав:
**r (read)** – чтение ( 4 )
**w (write)** – запись ( 2 )
**x (execute)** – исполнение ( 1 )

+ Права задаются для трёх категорий пользователей:
+ **u (user)** – владелец
**g (group)** – группа
**o (other)** – остальные пользователи

---

## Права для каталогов

| Право | Что означает для каталога |
|------|----------------------------|
| **r** | просмотр файлов в каталоге |
| **w** | создание и удаление файлов |
| **x** | вход в каталог и доступ к файлам |

> Права задаются для владельца, группы и остальных пользователей.
---

## Просмотр и изменение прав доступа:

+ Проверка прав доступа:
`ls -l`

+ Изменение прав доступа:
`chmod ( сумма цифр для выбора прав, где 0 - файл, 1 - директория ) file.name`

+ Изменение прав доступа рекурсивно:
`chmod -R ( сумма цифр ) directory.name`
> все файлы лежащие в директории, получат те же права, которые указаны были ранее.

> а ключ -c -> показывает сообщение об изменение файла;

+ Изменение владельца:
`chown user:group directory.name`

---

+ Символьный способ изменения прав доступа:

`chmod u=w,g=e,o=r file.name`  
> команда, где владельцу разрешено только писать, группе - исполнять, а остальным - читать;

`chmod a=e file.name` 
> команда, разрешающая всем только исполнять;

`chmod u-r,o+w file.name`
> команда, удаляющая права владельцу на чтение, но добавлению остальным прав писания;
