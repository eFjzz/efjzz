# Header format for TCP:

---

## Формат заголовка TCP:

| Поле                      | Описание                          |
| ------------------------- | --------------------------------- |
| **Source Port**           | Порт отправителя                  |
| **Destination Port**      | Порт получателя                   |
| **Sequence Number**       | Порядковый номер                  |
| **Acknowledgment Number** | Номер подтверждения               |
| **Data Offset**           | Смещение данных (длина заголовка) |
| **Flags**                 | Флаги управления                  |
| **Window**                | Размер окна                       |
| **Checksum**              | Контрольная сумма                 |
| **Urgent Pointer**        | Указатель на срочные данные       |
| **Options**               | Параметры (необязательно)         |
| **Data**                  | Данные (необязательно)            |

---

## Описание полей:

### Sequence Number (Порядковый номер)

> Указывает номер **первого байта** в сегменте

### Acknowledgment Number (Номер подтверждения)

> Кумулятивное подтверждение
> Указывает номер **следующего ожидаемого байта**

### Data Offset (Смещение данных)

> Показывает длину заголовка TCP

---

## Флаги TCP:

| Флаг    | Описание                              |
| ------- | ------------------------------------- |
| **CWR** | Управление перегрузкой                |
| **ECE** | Уведомление о перегрузке              |
| **URG** | Срочные данные присутствуют           |
| **ACK** | Подтверждение получено                |
| **PSH** | Передать данные приложению немедленно |
| **RST** | Принудительное завершение соединения  |
| **SYN** | Установка соединения                  |
| **FIN** | Завершение соединения                 |

---

## Window (Размер окна)

> Объем данных, которые получатель готов принять

---

## Checksum (Контрольная сумма)

> Используется для обнаружения ошибок при передаче

---

## Urgent Pointer (Указатель на срочные данные)

> Используется вместе с флагом **URG**

---

## Параметры TCP (Options):

| Параметр                            | Описание                                 |
| ----------------------------------- | ---------------------------------------- |
| **MSS (Maximum Segment Size)**      | Максимальный размер сегмента             |
| **Window Scale**                    | Масштабирование окна (до ~1 ГБ)          |
| **SACK (Selective Acknowledgment)** | Выборочное подтверждение диапазонов байт |
| **Timestamp**                       | Измерение времени доставки пакетов       |
