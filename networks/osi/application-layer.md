# Application Layer (OSI / TCP-IP)

## **Прикладной уровень (Application Layer)** 

+ это уровень сетевой модели, на котором работают **сетевые приложения**.

> Он обеспечивает **взаимодействие программ на разных устройствах через сеть**.

+ Примеры сетевых приложений:

- Web-браузеры
- Почтовые клиенты
- Мессенджеры
- FTP-клиенты
- DNS-клиенты

---

# Хосты (Hosts):

+ **Host (хост)** — это сетевое устройство, на котором запускаются сетевые приложения.

+ Примеры хостов:

- персональные компьютеры
- серверы
- смартфоны
- виртуальные машины

+ Главная роль хоста:

> запуск приложений, которые взаимодействуют друг с другом через сеть.

---

# Взаимодействие приложений

+ На **прикладном уровне** приложения взаимодействуют между собой **с помощью прикладных протоколов**.

+ прикладной уровень **изолирован от сетевого оборудования**
+ приложения **не знают**, как именно данные передаются по сети

> они просто отправляют данные через сетевые протоколы.

---

# Изоляция уровней

+ Протоколы прикладного уровня **не занимаются передачей данных по сети**.
+ Эти задачи выполняют **нижние уровни сети**.

---

## Связь с транспортным уровнем

+ Для передачи данных приложения используют **протоколы транспортного уровня TCP/IP**.

+ То есть:

`Application Layer
↓
Transport Layer
↓
Internet Layer
↓
Network Access Layer`

+ Для приложений создаётся иллюзия, что:

> программы на разных хостах обмениваются данными напрямую.

> хотя на самом деле данные проходят через множество сетевых устройств.

---

# Сервисы транспортного уровня

+ Транспортный уровень TCP/IP предоставляет **разные типы передачи данных**.

Основные протоколы:

- TCP
- UDP

---

# TCP (Transmission Control Protocol)

**TCP** — протокол надежной передачи данных.

Основные характеристики:

- устанавливает соединение между хостами
- обеспечивает **надежную передачу данных**
- гарантирует **порядок доставки**
- контролирует ошибки передачи
- передает **поток байтов**

TCP используется, когда важно:

- сохранить данные без потерь
- получить данные в правильном порядке

---

## Особенности TCP

+ TCP обеспечивает:

1. устанавливает сетевое соединение;
2. протокол управления передачей;
3. обеспечивает надежную передачу потока байтов - ( большая порция данных );

> поэтому TCP используется для передачи **больших объемов данных**.

---

## Примеры использования TCP

TCP используется в протоколах:

- HTTP
- HTTPS
- SMTP
- FTP
- SSH
