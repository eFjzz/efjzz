# Linux

+ **Linux** ( Bash )- это ядро операционной системы, в отличие от 
условного Windows ( PowerShell ) его особенности заключаются в том, что он:

1. бесплатный и безопасный;
2. разнообразие: большое кол-во дистрибутивов в открытом доступе;
3. высокая производительность и оптимизированность на более старом оборудовании. 
4. и, конечно же в открытом коде.

+ SSH ( Secure Shell ) - это протокол, который обеспечивает безопасное подключение к серверу через терминал;
+ DNS ( Domain Name System ) - система, позволяющая переводить узнать IP-адрес через домен;
+ SSL и TLS (Secure Sockets Layer / Transport Layer Security) - криптографические протоколы, обеспечивающие безопасную передачу данных;
а сертификаты - это файл, удостоверяющие подлинность сайта и обеспечивающие шифрование;

+ Bare-metal -  физический сервер в data centre, который обычно арендуют под собственное пользование на определенном железе.
+ Virtualization - технология создания виртуальных машин ( на 1 физ. сервере )  через разные приложения, например - 
+ VirtualBox - программа для создания виртуальных машин;

+ Slack - мессенджер для общения с командой со своими тредами, папками, отдельными чатами; ( без открытого доступа )

+ GitHub - платформа предназначеная для сгружения файлов с системы;
также можно сделать их в октрытом доступе для передачи или приватными,чтобы хранить собственные файлы и работы;

+ (!) Confluence - платформа созданая для компаний, которая делится базами знаний для своей команды;

+ (!) Jira - система для задач и тикетов;
+ (!) Zabbix, Grafana, Kibana - программы, предназначеные для мониторинга серверов и сетей; просмотру и анализу логов;

+ (!) SQL ( Structured Query Language ) - универсальный язык запросов к базам данных;
> она упрощает работу с огромными РБД ( реляционнами базами данных ); 

+ (!) FTP ( File Transfer Protocol ) - протокол для передачи файлов по сети; 
+ (!) SFTP ( Secure File Transfer Protocol ) - передача файлов через SSH;

+ HTTP's - тот же протокол, который использует SSL/TLS для шифрования данных.
+ HTTP ( Hypertext Transfer Protocol ) - протокол передачи данных в интернете; 
> ( используется для обмена данных между браузером и сервером / сервером и клиентом )

+ HTML ( Hypertext Markup Language ) - язык разметки для веб-страниц (он задает структуру сайта, где будет написан текст, где будет находится картинка и тп < >);
