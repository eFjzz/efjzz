### Будет преимуществом: 

!!! ( тех. собеседование ) - опыт работы с ОС семейства Linux (Ubuntu, CentOS, Debian);
!!! ( тех. собеседование ) - базовые навыки работы в командной строке Linux;
!!! ( тех. собеседование ) опыт установки ОС Windows\Linux с нуля.

- конкретно с установой Linux Ubuntu:

1. сначала скачиваем ISO-образ дистрибутива Ubuntu;
2. записываем образ на загрузочную флешку с помощью утилиты ( напр. Rufus )
3. перезагружаем компьютер и нам нужно попасть в Boot Menu или BIOS; 
( до того, как компьютер загрузит первоначальную ОС)
4. в меню выбираем загрузку с загрузочной-флешки или ставим её первой в Boot Priority; 
(чтобы загрузка системы производилась с загрузочной флешки)
5. после загрузки установщика выбираем язык и параметры установки.
6. затем настраиваем разметку диска (автоматическую или вручную, например ext4).
7. создаем пользователя и пароль, а также hostname (имя системы).
8. установщик копирует файлы системы на диск и устанавливает bootloader (обычно GRUB).
9. после завершения установки перезагружаем компьютер и уже загружаемся с установленной системой.
