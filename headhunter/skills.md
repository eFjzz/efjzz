# Linux
+ **Linux** ( Bash ) - это ядро операционной системы, в отличие от 
условного Windows ( PowerShell ) его особенности заключаются в том, что он:
1. бесплатный и безопасный;
2. разнообразие: большое кол-во дистрибутивов в открытом доступе;
3. высокая производительность и оптимизированность на более старом оборудовании. 
4. и, конечно же в открытом коде.
+ ***VirtualBox*** - программа для создания виртуальных машин;
> чтобы удаленно подключаться к разным ОС и ее дистрибутивам без сноса первоначальной ОС;
> Например: 1. Windows, 2. Linux, 3. MacOS и др.
+ Virtualization - технология создания виртуальных машин на физ. сервере через разные приложения.
> например - VMware или VirtualBox.

## Protocols:
+ ***TLS (SSL)*** 
> (Secure Sockets Layer / Transport Layer Security) 
+ криптографические протоколы, обеспечивающие безопасную передачу данных;
+ ***SSL-сертификат*** шифрует передачу данных между браузером и сервером, обеспечивая сохранность персональных данных ваших пользователей.
> сертификат - это файл, удостоверяющие подлинность сайта и обеспечивающие шифрование;
+ ***HTTPS*** - защищенный протокол ***HTTP***, который использует ***TLS (SSL)*** для шифрования данных.
+ ***HTTP*** ( Hypertext Transfer Protocol ) - протокол передачи (обмена) данных между сервером и браузером или сервером и клиентом; 
+ ***SSH*** ( Secure Shell ) - протокол, который обеспечивает безопасное подключение к серверу через терминал;
+ ***SFTP*** ( Secure File Transfer Protocol ) - защищенный протокол передачи файлов через SSH;
> FTP ( File Transfer Protocol ) - протокол для передачи файлов по сети; 
+ DNS ( Domain Name System ) - система, позволяющая переводить узнать IP-адрес через домен;

## Useful Platforms:
+ ***Confluence*** - платформа созданая для компаний, которая делится базами знаний для своей команды;
+ ***Slack*** - платформа для общения с командой ( со своими тредами, папками, отдельными чатами );
+ ***GitHub*** - платформа, предназначеная для сгружения файлов с системы ( через портал или терминал );
+ ***Jira*** - система для задач и тикетов;
+ ***Kibana*** - программа, используемая для поиска, анализа и визуализации логов (текстовых данных);
+ ***Zabbix*** - автономная **система мониторинга** серверов (сети или приложений). 
> собирает данные (CPU, RAM, сеть) и отправляет оповещения при сбоях. 
+ ***Grafana*** - платформа для создания дашбордов.  
> данные не собирает, а берет их из Zabbix;

## Languages of IT:
+ ***SQL*** ( Structured Query Language ) - универсальный язык запросов к базам данных;
> она упрощает работу с огромными РБД ( реляционнами базами данных ); 
+ ***HTML*** ( Hypertext Markup Language ) - язык разметки для веб-страниц;
> он задает структуру сайта, где будет написан текст, где будет находится картинка и тп `</h1>`;
